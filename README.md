# Digital Memory Capsule

A cinematic memory journaling web application built using React, Vite, Tailwind CSS, and Framer Motion.

## ✨ Features

- Animated cinematic UI
- Interactive memory timeline
- Dynamic mood-based themes
- Persistent storage using localStorage
- Responsive design for mobile and desktop
- Smooth animations and cursor glow effects

## 🛠️ Tech Stack

- React
- Vite
- Tailwind CSS
- Framer Motion
- JavaScript
- HTML5
- CSS3

## 🚀 Live Demo

https://digital-memory-capsule-wheat.vercel.app/
